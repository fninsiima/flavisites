The WHITEFLY Counter program detects and counts whiteflies on infested leaves.

The File menu can be used to load existing images, and to exit the application.

All the most commonly used menu options are also available on the toolbar. The toolbar also provides an easy way of opening the image.

To use the drag and drop feature
Simply drag and drop the image into the application.
Image files are automatically saved with "_detected" added to the filename in a folder called results that is created when the application is run. For example:
imagename_detected.jpg

To use the application
Click Browse to select the folder with the images
To score the images automatically, click the Score automatically button
A progress bar with notify you of completion and you will be able to download a CSV file or go to the folder
To score the images semi-automatically, browse to the selected folder and click the semi-automatic button
Use the Previous or Next buttons to navigate between images
Image files are automatically saved with "_detected" added to the filename in a folder called data that is created when the application is run.
